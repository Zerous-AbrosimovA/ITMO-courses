# Исключения

```C++
try
{
  ...
  if(error)
    throw 4;
  ...
}
catch(int e)
{
  ...
}
catch(int *e)
{
  ...
  throw e;
}
```

## Использование исключений для new

```C++
try
{
  S *p = new S;
}
catch(...) // ловить все ошибки
{
  printf("error");
  throw;
}
```

В 64 битных системах try-catch почти бесплатный, а в 32 битных требуется ресурсов

```C++
uint div(uint a, uint b)
{
  if (b == 0)
    throw "division by zero";
  return a / b;
}
```
